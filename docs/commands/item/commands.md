## item.getName
Get an item's name.
### Command Parameters
- **itemStack** Item to check
### Return Type
Returns itemStack.

## item.load
Get a value of a symbol from the item.
### Command Parameters
- **itemStack** Item to manipulate
- **symbol** Symbol to get value of
### Return Type
Returns any.

## item.setLore

- **itemStack** Item to manipulate
- **list[string]** Lore to set
### Return Type
Returns itemStack.
