## entity.summon
Spawn an entity in the world. Returns it's UUID.
### Command Parameters
- **string** Type of entity to spawn
- **:loc** Location to spawn the entity
### Return Type
Returns string.

## entity.summonItem
Spawn a dropped item in the world. Returns it's UUID.
### Command Parameters
- **itemStack** Item to spawn
- **:loc** Location to spawn the entity
### Return Type
Returns string.
